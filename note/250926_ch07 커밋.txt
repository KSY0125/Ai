7장. 객체지향 프로그래밍
	1절. 객체와 클래사
	  class 클래스명 (상속 받을 클래스명):
    	  "독스트링" > 함수 생성시와 똑같음
       	  이 곳에 변수나 함수들을 선언
	2절. 생성자와 소멸자
	 - 생성자 : __init__(self, [매개변수1, 매개변수2..]) - 객체가 생성될 때 자동 실행
	 - 소멸자 : __del__(self) - 객체가 소멸될 때 자동 실행
	3절 변수와 메서드(함수)
	 - 인스턴스 변수 vs 클래스변수 스태틱변수(정적변수)
	 - 인스턴스메서드(self) vs 클래스메서드(cls) vs static메서드()
	    **스태틱메서드는 정적메서드라고 해석하진 않음
	4절 상속과 재정의
    	 -  문법 형태 : class자식클래스명(부모클래스명):
            	~~~함수 뭐시기 뭐시기~~~
	 - 상속을 '클래스 재사용'이라고도 함
	 - 상속을 이용하면 부모클래스의 모든 속성을 자식클래스로 물려줄 수 있음. 
	    >>부모가 하는 게 없고 자식이 움직임.
	 - 상속 시 init을 제외한 것들만 상속됨.
	 - super() : 부모클래스의 세팅값을 자식클래스로 불러옴
	 - 스태틱 변수(정적변수)
	    >>보통, static 변수 : 앞에 under score 2개(__) 붙힘.
	    >>여러 객체들 사이에 데이터를 공유하고 싶을 때 사용
	    >>클래스명._클래스명__static변수명으로 참조
